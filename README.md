# 프록시 서버

http 통신에 있어서 작동하는 포워드 프록시 서버를 개발했습니다.

## 기능

* 특정 패턴이 url에 들어갈 때 지정된 주소로 리다이렉트
* 이미지 전송을 거부하는 쿼리

## 작동 화면

1. 특정 패턴(korea)가 url에 들어갈 경우 지정된 주소(http://mnet.yonsei.ac.kr/)로 리다이렉트되는 기능
